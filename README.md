# ChatDOC_API_Demo

## Environment

- Node.js >= 16.x
- pnpm >= 8.x

## Installation

run `pnpm install`

## Custom server environment

- Set the env variable PROXY_API_KEY 

  > **PROXY_API_KEY**  is the API key that you obtained after purchasing the API package or membership plan.


## Getting started

run `pnpm start`
